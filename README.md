## 'top' 명렁어
### 1. 개요

> top - 실행 중인 시스템 상태를 **실시간으로** 모니터링하는 명령어
- CPU, 메모리 사용량, 실행 중인 프로세스 목록을 **실시간으로** 갱신해서 보여줌.

### 2. 주요 출력 항목
`top` 실행 시 상단부(요약 정보):
- `uptime` : 시스템이 켜져 있는 시간
- `Tasks` : 전체/실행/대기/중지/좀비 프로세스 수0
- `%Cpu(s)` : CPU 사용 비율 (us, sy, ni, id, wa, hi, si 등) 
- `MiB Mem` : 실제 메모리 사용량
- `MiB Swap` : 스왑 영역 사용량

`top` 실행 시 하단부(프로세스 목록):
- PID, USER, PR, NI, VIRT, RES, %CPU, %MEM, TIME+, COMMAND 등

### 3. 유용한 옵션
- `-b` : **batch 모드** -> 실시간 인터렉션 없이 출력만 하도록 함.
- `-p` : 특정 PID만 모니터링함.
- `-1` : CPU 코어별 사용량 표시
- `-k` : 선택한 PID에 신호 보내기
- `-r` : 프로세스 nice값(우선순위) 변경

## 'ps' 명령어
### 1. 개요

> ps - 현재 실행 중인 프로세스들의 상태를 스냅샷 형태로 출력하는 명령어

- 스크립트나 로그 확인을 할 때 유용하다.
